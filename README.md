# CMake Aubio

Build [Aubio](https://aubio.org) as a static library using CMake.

## Build :building_construction:

```bash
git clone --recursive https://github.com/rghvdberg/aubio_cmake.git
cd aubio_cmake
mkdir build
cd build
cmake ../
make
```
